# Bootstrap 5 Exercises

## 1. Setting Up Bootstrap 5

### Exercise 1.1: Bootstrap via CDN

```html
<!DOCTYPE html>
<html>
<head>
<title>Bootstrap CDN</title>
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>

<div class="container mt-5">
<h1 class="text-primary">Bootstrap 5 CDN Connected</h1>
</div>

</body>
</html>
```

### Exercise 1.2: Bootstrap Using Downloaded Files

```html
<link rel="stylesheet" href="css/bootstrap.min.css">
<script src="js/bootstrap.bundle.min.js"></script>
```

## 19. Customization with Sass

### Exercise 19.1

Install Bootstrap and Sass

```bash
npm install bootstrap
npm install sass
```

Import Bootstrap

```scss
@import "bootstrap/scss/bootstrap";
```

### Exercise 19.2

```scss
$primary: #ff5722;
$border-radius: 1rem;

@import "bootstrap/scss/bootstrap";
```

Compile Bootstrap

```bash
sass custom.scss custom.css
```
